# slidev-theme-whu-landing

[![NPM version](https://img.shields.io/npm/v/slidev-theme-whu-landing?color=3AB9D4&label=)](https://www.npmjs.com/package/slidev-theme-whu-landing)

A [Slidev](https://github.com/slidevjs/slidev) theme for WHU MARS report.


## Install

Add the following frontmatter to your `slides.md`. Start Slidev then it will prompt you to install the theme automatically.

<pre><code>---
theme: <b>whu-landing</b>
---</code></pre>

Learn more about [how to use a theme](https://sli.dev/guide/theme-addon#use-theme).

## Contributing

- `pnpm install`
- `pnpm run dev` to start theme preview of `slides/main.md`
- Edit the `slides/` and style to see the changes
- `pnpm run export` to generate the preview PDF
- `pnpm run screenshot` to generate the preview PNG
